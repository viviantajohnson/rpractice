# The LSD Lab Guide to R

## Welcome to the Language & Social Dynamics (LSD) Lab R Guide!  
This repository provides everything a new R user needs to get up and running, including installing the software,learning the RStudio interface, and basic analyses like descriptive statistics, correlations, t-tests, and simple data transformations.
